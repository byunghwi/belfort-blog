### 구현 목적 및 방법

**Why**:  **공간 정보 오픈플랫폼 지도서비스**인 **V-World**를 사용하여 3D정밀 지도를 표출하고, 
**3D 지리 정보 시각화를 위한 오픈 소스 JavaScript 라이브러리**인 **Cesium.js**을 사용하여 3D모델링 표출 및 시뮬레이션을 하기 위함

**How:  `V-World 웹지엘 3D지도 API 3.0`** 를 활용하여 3D지도 표출 및 API 3.0에서 추가된 Cesium 확장 기능을 사용
